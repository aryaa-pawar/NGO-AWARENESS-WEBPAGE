# InAmigos Foundation Awareness Webpage

This project is a basic NGO awareness webpage created for InAmigos Foundation. It highlights the NGO's purpose, ongoing initiatives, social impact, visuals, and ways for people to get involved.

## Project Objective

The objective of this webpage is to spread awareness about InAmigos Foundation projects and encourage visitors to support, volunteer, or connect with the organization.

## Sections Included

- Introduction about InAmigos Foundation
- Details of ongoing projects
- Social impact and purpose of the initiatives
- Image gallery with project visuals
- Call-to-action section for joining, volunteering, or supporting
- Contact information and quick connect form

## Technologies Used

- HTML
- CSS
- Font Awesome icons

## How to Open

Open the `index.html` file in any web browser.

## Files

- `index.html` - Main webpage structure
- `style.css` - Styling and responsive layout
- `images/` - Logo and project visuals
